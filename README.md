# ProyectoFinalUnidad1

## GunsfierroCund- Prototipo de Sitio Web

## Descripción del Proyecto

Este es un prototipo de sitio web desarrollado para la materia de Desarrollo Web. El sitio presenta información sobre armas de fuego (pistolas, fusiles y ametralladoras) con un diseño moderno y responsivo utilizando **Bootstrap 5**.

El proyecto demuestra la implementación de:
- Barra de navegación con redirección entre páginas
- Carrusel de imágenes interactivo
- Sistema de columnas responsivo
- Formularios de contacto
- Footer estructurado
- Diseño adaptable a móvil, tablet y escritorio

## Estructura del Proyecto

Páginas Principales
index.html
Barra de navegación: Inicio y Página 2.

Carrusel: 3 imágenes con controles y leyendas.

3 secciones de contenido (columnas responsivas):

Imagen + lista de pistolas
Video (YouTube) sobre fusiles
Formulario de contacto
Footer: 3 columnas (creador, redes sociales, enlaces de interés).

pagina2.html
Misma barra de navegación (activo "Página 2").

Catálogo en cards: organizado en filas de 3 cards (pistolas, fusiles, ametralladoras), cada una con imagen, título y lista de características.

Footer idéntico para consistencia.

## Responsive Design
Escritorio (≥992px): 3 columnas

Tablet (768px - 991px): 2 columnas

Móvil (<768px): 1 columna

Menú colapsable en móviles

Carrusel con altura ajustable (500px → 300px)

Estilos Personalizados (iniciostyle.css)
Altura fija para el carrusel y ajuste de imágenes (object-fit: contain)

Efecto hover en cards (elevación)

Estilos para títulos y espaciado


## Instalación y Ejecución

Descargar o clonar el repositorio.

Abrir index.html en cualquier navegador moderno.

Navegar mediante la barra de navegación.
